## 📋 Checklist de Preenchimento

Antes de enviar, valide:

- [ ] **Situação Atual:** Dor específica + custo quantificado
- [ ] **Resultado Desejado:** Transformação clara + valor quantificado
- [ ] **Core Product:** Features → Benefits (não apenas lista técnica)
- [ ] **Bônus (mínimo 2):** Cada um remove objeção ou amplifica resultado
- [ ] **Valor Total:** 1.5-3x o investimento final
- [ ] **Urgency:** 2-3 razões REAIS para agir agora
- [ ] **Scarcity:** Real (não fake)
- [ ] **Guarantee:** Forte (reverse risk)
- [ ] **Cases:** Mínimo 2 depoimentos reais (nome completo + resultado)
- [ ] **Cronograma:** Específico (não "em breve")
- [ ] **Validade:** Data clara de expiração
- [ ] **Revisão:** Zero erros de português/digitação

## 💡 Dicas de Uso

### ✅ FAÇA:
- Personalize cada seção para o cliente específico
- Use números reais (não "muito", "bastante", "cerca de")
- Foque em benefícios (não apenas features técnicas)
- Seja específico: "Economiza 30h/mês" >> "Economiza tempo"
- Mantenha empático mas objetivo (sem enrolação)

### ❌ NÃO FAÇA:
- Copiar/colar genérico (cliente percebe)
- Fake scarcity ("só hoje", mas repete todo dia)
- Garantia fraca ("faremos nosso melhor")
- Cases vagos ("cliente X ficou satisfeito")
- Excesso de texto (cliente quer clareza, não romance)
