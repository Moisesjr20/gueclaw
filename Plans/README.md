# 📁 Plans — Planejamentos DOE

Esta pasta contém todos os planos de projeto gerados pelo GueClaw usando o framework **DOE** (Directives, Orchestration, Execution).

---

## Estrutura

```
Plans/
├── README.md                        ← este arquivo
├── <nome-do-projeto>/
│   ├── PLANO.md                     ← Documento de planejamento completo
│   └── CHECKLIST.md                 ← Checklist operacional de execução
└── ...
```

---

## Convenções

| Convenção | Regra |
|---|---|
| Nome do diretório | `kebab-case` do nome do projeto |
| `PLANO.md` | Documento estratégico: objetivo, fases, riscos, recursos |
| `CHECKLIST.md` | Documento operacional: tarefas diárias, validações, bloqueadores |
| Tarefas | Sempre com verbo no infinitivo, claras e acionáveis |
| Status do plano | 🟡 Planejamento / 🟢 Em execução / ✅ Concluído / 🔴 Bloqueado |

---

## Como pedir um plano

Basta dizer ao GueClaw:
- "Quero planejar [projeto]"
- "Cria um plano de ação para [objetivo]"
- "Me ajuda a organizar o projeto [nome]"
- "Monta um roadmap para [entrega]"

O GueClaw vai coletar as informações necessárias e gerar os dois arquivos automaticamente nesta pasta.
